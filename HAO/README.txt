How to run
===========
1) Ensure you have Python 3.10+ and run:
   pip install -r requirements.txt

2) Start the app from this folder by:
   python main.py

3) The Flask app will start at http://127.0.0.1:5000

Notes
-----
- The web package is in ./website
- A working SQLite database with demo data is in ./website/app.db (not required by the JSON code path but included per assignment).
- Do NOT add any virtualenv folders into the zip.
